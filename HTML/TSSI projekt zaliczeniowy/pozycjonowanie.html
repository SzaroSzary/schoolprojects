Pozycjonowanie

Za pozycjonowanie odpowiedzialna jest właściwość position. Może ona przyjmować następujące własności:

  1. static - statyczne (domyślne)
    • element blokowy zajmuje całą dostępną szerokość
    • wysokość elementu blokowego jest ustalana na podstawie zawartości, jest to najmniejsza możliwa wartość, dla której treść zmieści się wewnątrz elementu
    • elementy blokowe, które nie są zagnieżdżone ukłądają się jeden pod drugim
    • jeśli zagnieżdżamy jeden element blokowy w drugim, to obszar zajmowany przez dziecko będzie leżał wewnątrz obszaru zajmowanego przez rodzica
  2. relative - względne
    • element ma taki sam wpływ na pozycjonowanie innych elementów jak w statycznym
    • właściwości left, right, bottom i top przesuwają element
    • przesunięcie elementu obliczane jest względem jego lewego górnego narożnika
  3. absolute - bezwzględne
    • element pozycjonowany bezwzględnie nie ma wpływać na inne elementy
    • położenie elementu zostanie ustalone na podstawie bieżącego kontekstu, domyślnym kontekstem pozycjonowania jest lewy górny narożnik strony
    • właściwości left, right, bottom i top przesuwają element
  4. fixed - trwałe
    • element nie podlega przewijaniu
    • element pozycjonowany trwale nie ma wpływać na inne elementy
    • położenie elementu zostanie ustalone na podstawie bieżącego kontekstu, domyślnym kontekstem pozycjonowania jest lewy górny narożnik strony
    • właściwości left, right, bottom i top przesuwają element
  5. względnie bezwzględne
    • dotyczy elementów zagnieżdżonych
    • element nadrzędny musi mieć pozycjonowanie względne (position: relative;)
    • element podrzędny musi mieć pozycjonowanie bezwzględne (position: absolute;)

Do przydatnych właściwości CSS przy pozycjonowaniu należą:

  • z-index - ustala kolejność warstw na stronie od warstwy najbardziej z tyłu
  • border-radius - dodaje zaokrąglone rogi
  • overflow - określa co się stanie z zawartością bloku elementu, kiedy przekroczy
    ona jego granice
  • letter-spacing - dodaje odstępy pomiędzy literami
  • font-variant: small-caps; - dodaje kapitaliki
  • text-indent - dodaje wcięcie dla pierwszego wiersza akapitu

Należy pamiętać, że jeśli ustawimy w znaczniku <footer> position: fixed; to trzeba zdefiniować również jego szerokość. Powinno się również unikać umieszczania właściwości float i position jednocześnie.
